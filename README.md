# Dragoncraft

A peaceful medieval fantasy village-building and dragon-nurturing strategy game.

## Description

Build and manage a medieval village while nurturing dragons from egg to adulthood. Inspired by the style of Age of Empires II, Dragoncraft blends resource management, village simulation, and dragon lifecycle gameplay into a relaxed strategy experience.

## Engine

- **Engine:** Godot 4
- **Language:** GDScript

## Status

In development.

## Key Systems

- **World Map** — navigate between villages and regions
- **Village Simulation** — construct buildings, manage villagers and production
- **Dragon Lifecycle** — hatch eggs, raise hatchlings, bond with adult dragons
- **Resource Management** — gather and spend wood, stone, food, and arcane resources
